a. ¿Quién inventó el sistema de control de versión Git y por qué?

Git es un es un software de control de versiones desarrollado ginalmente Linus Torvalds, 
el famoso creador del kernel del sistema operativo Linux, en 2005.
Linus Torvalds buscaba un sistema distribuido que pudiera usar en forma semejante a BitKeeper, 
pero ninguno de los sistemas bajo software libre disponibles cumplía con sus requerimientos, especialmente en cuanto a desempeño.
